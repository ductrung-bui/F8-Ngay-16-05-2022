Ví dụ comments trong HTML

Cập nhật tháng 2 năm 2022

  1. Hãy mở tab TRÌNH DUYỆT và xem kết quả được hiển thị
  2. Hãy thử thay đổi nội dung file index.html để xem kết quả (VD. Bỏ cú pháp comments hoặc thêm cú pháp comments)

Hãy nhớ phím tắt:

  • Windows: Ctrl + /
  • MacOS: Cmd + /

Cú pháp comments để ghi chú trong code, nó không được hiển thị ra trình duyệt, hoặc có thể sử dụng để tạm vô hiệu hóa một số dòng code.
